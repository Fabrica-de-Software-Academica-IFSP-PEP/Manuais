    Cardápios

    Visão Geral

    A tela de Gestão de Cardápios é o módulo central do Sistema de Gestão Nutricional (SGN), responsável pelo planejamento e organização das refeições. Esta interface permite criar, visualizar, editar e gerenciar cardápios semanais completos, com cálculos nutricionais automáticos e integração total com o banco de Pratos cadastrados.

    Características da Interface

    Layout e Navegação

    • Barra superior verde: Menu principal com acesso a todos os módulos do sistema
    • Botão Voltar: Navegação intuitiva para retornar à tela anterior
    • Design responsivo: Interface adaptável para diferentes tamanhos de tela
    • Identidade visual consistente: Seguindo o padrão verde do SGN

    Área Principal de Dados

    A interface apresenta uma tabela organizada com as seguintes colunas:

    Informações Exibidas

    • CARDÁPIO: Nome e descrição do cardápio planejado
    • PERÍODO: Data de início e fim de vigência
    • KCAL DIÁRIA: Valor calórico total planejado por dia
    • STATUS: Situação atual (Ativo, Inativo, Em Planejamento)
    • AÇÕES: Botões para visualizar, editar, clonar e gerar relatórios

    Funcionalidades de Busca e Filtros

    • Campo de busca: "Buscar por Cardápio" para filtrar registros rapidamente
    • Filtro por período: Seleção de data de início e fim
    • Filtro por status: Exibição de cardápios ativos, inativos ou em planejamento
    • Busca em tempo real: Resultados atualizados conforme digitação

    Ações Disponíveis

    • Novo Cardápio: Botão verde no canto superior direito
    • Visualizar: Ícone de olho para ver detalhes completos
    • Editar: Ícone de lápis para modificar informações
    • Clonar: Ícone de cópia para duplicar cardápios existentes
    • Compartilhar: Ícone de link para gerar URLs de compartilhamento
    • Relatórios: Acesso a relatórios detalhados e impressões

    Dados dos Cardápios

    Informações Nutricionais

    Cada cardápio possui cálculos automáticos de:

    • Valor calórico diário total
    • Distribuição de macronutrientes (carboidratos, proteínas, gorduras)
    • Micronutrientes essenciais (vitaminas e minerais)
    • Adequação nutricional por faixa etária
    • Análise de custos dos ingredientes

    Estrutura Temporal

    • Períodos personalizáveis: De dias a semanas completas
    • Refeições organizadas: Café da manhã, almoço, lanche e jantar
    • Cardápio do dia: Destaque para o menu atual
    • Histórico completo: Registro de todos os cardápios anteriores

    Funcionalidades Principais

    Criação de Cardápios

    • Assistente de criação: Processo guiado para novos cardápios
    • Seleção de pratos: Integração com biblioteca de receitas cadastradas
    • Cálculo automático: Valores nutricionais computados em tempo real
    • Validação nutricional: Verificação de adequação às diretrizes
    • Definição de períodos: Configuração flexível de datas

    Visualização Detalhada

    • Visão semanal: Grade completa com todas as refeições
    • Detalhes nutricionais: Informações completas de cada dia
    • Lista de ingredientes: Todos os alimentos utilizados
    • Modo de visualização pública: Interface para consulta dos usuários

    Edição e Atualização

    • Modificação de pratos: Alteração de receitas em qualquer refeição
    • Ajustes nutricionais: Otimização para atingir metas específicas
    • Substituições inteligentes: Sugestões de alternativas equivalentes
    • Versionamento: Controle de alterações com histórico
    • Aprovação: Sistema de validação por nutricionistas

    Clonagem de Cardápios

    • Duplicação completa: Cópia de cardápios existentes
    • Adaptação de períodos: Ajuste automático de datas
    • Modificações pontuais: Alterações específicas em cardápios clonados
    • Reutilização eficiente: Aproveitamento de planejamentos anteriores
    • Templates: Criação de modelos reutilizáveis

    Relatórios e Análises

    • Relatório semanal: Visão completa do planejamento
    • Análise nutricional: Detalhamento de macro e micronutrientes
    • Lista de compras: Quantidades necessárias de cada ingrediente
    • Relatório de custos: Análise financeira detalhada
    • Exportação em PDF: Documentos prontos para impressão

    Compartilhamento

    • Links públicos: URLs para consulta externa
    • Visualização sem login: Acesso público aos cardápios
    • QR Codes: Códigos para acesso rápido via dispositivos móveis
    • Integração com murais: Exibição em totens e painéis digitais
    • Notificações: Alertas sobre novos cardápios disponíveis

    Cardápio do Dia

    • Destaque especial: Menu atual em evidência
    • Informações imediatas: Refeições do dia facilmente visíveis
    • Status em tempo real: Atualização automática diária
    • Acesso rápido: Consulta instantânea para usuários
    • Detalhes nutricionais: Informações completas disponíveis

    Integração com Outros Módulos

    • Alimentos: Base de dados nutricional completa
    • Pratos: Receitas e preparações cadastradas
    • Compras: Geração automática de listas de ingredientes
    • Relatórios: Análises nutricionais e financeiras
    • Usuários: Controle de acesso e permissões

    Benefícios do Módulo

    Eficiência no Planejamento

    • Automatização de cálculos: Valores nutricionais computados automaticamente
    • Reutilização inteligente: Clonagem e adaptação de cardápios existentes
    • Validação automática: Verificação de adequação nutricional
    • Processo otimizado: Fluxo de trabalho simplificado para nutricionistas
    • Redução de erros: Minimização de falhas no planejamento

    Controle Nutricional Rigoroso

    • Precisão nos cálculos: Dados nutricionais confiáveis e atualizados
    • Adequação às normas: Conformidade com diretrizes nutricionais oficiais
    • Rastreabilidade completa: Histórico de todos os planejamentos
    • Monitoramento contínuo: Acompanhamento da qualidade nutricional

    Transparência e Comunicação

    • Acesso público: Consulta aberta aos cardápios planejados
    • Compartilhamento fácil: Link público para divulgação
    • Informação clara: Apresentação organizada e compreensível
    • Atualização em tempo real: Informações sempre atualizadas

    Dica: Utilize a função de clonagem para aproveitar cardápios bem-sucedidos anteriores, fazendo apenas os ajustes necessários. Isso economiza tempo e mantém a qualidade nutricional já validada.
