    Manual do Sistema de Gestão Nutricional - IFSP PEP

    Bem-vindo ao Manual do Sistema de Gestão Nutricional (SGN) do IFSP - Campus Presidente Epitácio!

    Este sistema foi desenvolvido para modernizar e integrar todos os processos da gestão nutricional institucional, oferecendo uma solução completa que abrange desde o cadastro de alimentos até a prestação de contas, proporcionando eficiência e transparência em todas as etapas.

    Primeiros Passos...

    1. Comece pela Introdução

    Leia a Introdução para compreender a arquitetura e objetivos do sistema.

    2. Explore os Módulos

    Familiarize-se com cada módulo através deste manual, seguindo a ordem do fluxo operacional.

    3. Configuração Inicial

    Configure primeiro: Fornecedores → Contratos → Alimentos → Produtos

    4. Operação Diária

    Utilize: Cardápios → Pedidos → Controle de Estoque → Relatórios

    Navegação neste Manual

    Utilize o índice à direita para navegar rapidamente entre as seções desta página. Cada módulo do sistema possuirá sua própria seção detalhada com:

    • Objetivos e funcionalidades
    • Passo a passo das operações
    • Capturas de tela explicativas
    • Dicas e boas práticas
    • Solução de problemas comuns

    Dica: Este manual é atualizado continuamente. Para dúvidas específicas ou sugestões de melhorias, entre em contato com a equipe de desenvolvimento.
